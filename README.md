## readNewspaper
自动获取电子版报纸，方便每天阅读

## 目前可以获取的报纸
* 《半岛都市报》  
报纸首页：http://bddsb.bandao.cn/

## 特性
-[x] 自动合并PDF
-[x] 代理IP池
-[ ] 运行脚本时自动获取代理IP池
-[ ] 通过参数选择对应日期的报纸

## 依赖
* PyPDF2
* requests
* lxml
